# 支持的快捷键 及 语法格式

# 快捷键

加粗：      Ctrl/Cmd + B
标题：      Ctrl/Cmd + H
插入链接：  Ctrl/Cmd + K
插入代码：  Ctrl/Cmd + Shift + C
行内代码：  Ctrl/Cmd + Shift + K
插入图片：  Ctrl/Cmd + Shift + I
无序列表：  Ctrl/Cmd + Shift + L
撤销：      Ctrl/Cmd + Z

# 语法

# 标题

语法格式：** '#'+'空格'+'文本' **

# 列表

无序列表语法格式：** '-' + '空格' ＋ '文本' **

有序列表语法格式：** '数字' + '.' + '空格' + '文本' **

任务列表语法格式：** '-' + '空格' + '[ ]' + '文本' **

# 链接和图片

插入链接，［显示文本］(链接地址)。 例如 稀土掘金

插入图片，![图片的标注](图片链接地址)

# 引用

语法：** '>'+'空格'+'文本' **

# 代码块

 语法：
 '```'
 '```'
