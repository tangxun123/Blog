# markdown快捷键及语法

# 快捷键

加粗：      Ctrl/Cmd + B
标题：      Ctrl/Cmd + H
插入链接：  Ctrl/Cmd + K
插入代码：  Ctrl/Cmd + Shift + C
行内代码：  Ctrl/Cmd + Shift + K
插入图片：  Ctrl/Cmd + Shift + I
无序列表：  Ctrl/Cmd + Shift + L
撤销：      Ctrl/Cmd + Z

# 语法

# 标题

语法格式：# + 空格 + 文本

# 列表

无序列表语法格式： - + 空格 + 文本

有序列表语法格式：数字 + . + 空格 + 文本

任务列表语法格式：- + 空格 + [ ] + 文本

# 链接和图片

插入链接，［显示文本］(链接地址)。 例如 稀土掘金

插入图片，![图片的标注](图片链接地址)

# 引用

语法：> + 空格 + 文本 ``

# 表格

Markdown　Extra　表格语法：

项目 价格
iPhone $560
iPad $780
iMac $1000

可以使用冒号来定义对齐方式：

项目 价格 数量
iPhone 6000 元 5
iPad 3800 元 12
iMac 10000 元 234
